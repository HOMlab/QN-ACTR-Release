Step 1 - Classes

__________________________

 

1 - The number of classes of the model can be specified in the corresponding input area

 

 

Note: The number of classes can be described using the keyboard or using the mouse, through the spin controls.

 

 

2 - In order to personalize a model classes may be given names other than default ones.

 

 

Default class names are: Class1, Class2...

 

 

3 - Definition of class types (Open or Closed) of the model can be described in this window

 

 

 

4 - The closed classes workload is described by the number of customers in each class.

The open classes workload is described by the customer arrival rate for each class.

Note:

Input boxes of the "Arrival Rate" group referring to open classes accept positive real numbers.

Input boxes of the "Customers" group referring to closed classes accept only positive integer numbers.

Load dependent models must have a number of customers greater than zero.
