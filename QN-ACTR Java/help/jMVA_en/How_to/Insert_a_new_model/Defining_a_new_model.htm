Definition of a new model

_________________

 

To define a new model select fast choice button   or the New command from File menu.

 

 

The following parameters must be introduced in order to define a new model:

 

Step 1

Classes of the model

 

Step 2

 

Stations

 

Step 3

 

Service Demands

or

Service Times and Visits

 

Step 4

 

Input an optional short comment

 

Step 5

 

Solve, model resolution
