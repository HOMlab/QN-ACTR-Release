Step 2 - Stations

____________________________

 

1 - The number of stations of the model can be specified in the corresponding input area

 

 

 

2 - In order to personalize a model stations and classes may be given names other than default ones.

Default resource names are: Station1, Station2...

 

 

3 - Definition of stations types (Load Independent, Load Dependent or Delay) of the model can be described in this window.

 

 

If station is "Load Dependent" a set of service demands must be defined (one service demand for each possible population)
