Relationship between input data of the model

____________________

 

Define:

Vk,c average number of visits to service resource k for class c customers

Sk,c average service time per visit to service resource k for class c customers

Dk,c average service demand of class c customers at service resource k

Service demand is given by

 

  Dk,c = Vk,c* Sk,c

The value of Dk,c is obtained in the following way:

 

1. When the value in one of the "Visits" input boxes of the dialog window is changed, the corresponding "Service Demands" input box will automatically be updated with the result of the product of the visits value by the service times value

2. When the value in one of the "Service Times" input boxes of the dialog window is changed, the corresponding "Service Demands" input box will automatically be updated with the result of the product of the visits value by the service times value

3. When the values in one of the "Service Demands" input boxes of the dialog window is changed, the corresponding "Service Times" input box will automatically be updated with the result of the division of the service demands value by visit value

 

Note

The values in the "Service Demands" input boxes must be greater than zero.
