Start simulation (Simulation menu)

_________________

 

Use this command to start the execution of the simulation. In case of errors, e.g., a missing component or wrong connections, a diagnostic box displays the error.
Simulation results are kept in memory and can be displayed at any time by clicking the Results button in the Simulation menu.
If the simulation is restarted, without previously saving the results, JSIM warns that results will be overwritten if the simulation continues.

 

 

Keyboard shortcut: CTRL+M
