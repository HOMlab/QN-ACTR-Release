Step 5 - Performance Indices

_________________

 

The Performance Indices tab allows you to define which perfomance indices to collect and plot.

 

Select the performance index from the the scroll menu below

 

 

then click the button to actually add the selected index (you can add as many indices as you wish).
In the Performance Index table it is now possible to specify for which class and for which station each perfomance index must be computed and plotted.
The Confidence Interval and Maximum Relative Error can also be specified to values of choice (default values are 0.9 and 0.1, respectively).

 

 

To remove a performance index from the table, click the corresponding button.
