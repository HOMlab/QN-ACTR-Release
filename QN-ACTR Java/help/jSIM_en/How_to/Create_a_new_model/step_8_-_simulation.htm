Step 8 - Simulation

_________________

 

The Simulation tab allows you to define the simulation run parameters and the properties of the initial state.

 

Simulation parameters, namely the seed of the pseudo-random number generator, the maximum duration in seconds, the maximum number of samples, and the time interval of animation refresh, can be specified. Although possible, it is recommended not to change the default seed, as it can compromise the quality of the results.

The initial state of the model is given by the number of customers at each station when the simulation starts.

 

The simulation is now ready to start, if a single point run is what you need. In this case, click the Start Simulation button in the command bar

 

On the contrary, if you would like to run a series of simulations where one or more parameters, such as number of customers in the system or service time of some stations vary over a user defined range, proceed to Step 9.
