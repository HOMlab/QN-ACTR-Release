Step 9 - What-if analysis

_________________

 

The What-if analysis tab allows you to define the parameters to vary for a series of consecutive simulation runs.

INSERIRE QUI LO SCREENSHOT DEL TAB WHAT IF COME APPARE QUANDO LO SI SELEZIONA

 

With the What-if analysis, a series of simulation runs can be executed varying one parameter, in particular the total number of jobs (or customers) in the system or the proportion of each class of jobs in the system (population mix) for models with closed classes, the jobs arrivale rate for models with open classes, the service time at a station for one or all the classes, or the simulation seed.
The What-if analysis must be explicitly enabled by checking the Enable box.

IMMAGINE DELLA TENDINA APERTA CON TUTTI I POSSIBILI PARAMETRI DI WHAT-IF

The number of customers can be varied over a range at the requested step for a specific class (see the picture below) or for all the classesat the same time. The initial customer distribution is described as Population mix. The data of the class(es) whose population will be varied appears in red.

IMMAGINE DELLA WHAT-IF ABILITATA PER NUMBER OF CUSTOMERS IN UN SISTEMA CON ALMENO 2 CLASSI CHIUSE, AUMENTARE I JOB IN ENTRAMBE LE CLASSI ABILITATO.

The population mix can be varied when the model has only two closed classes. In this case, it is possible to specify how to change the percentage of customers in one of the two classes. The other one is computed by complementation.

IMMAGINE DELLE POPULATION MIX CON UNA CLASSE CHE PARTE DA .1 E ARRIVA A .9

The service time of a station for one specific class, or for all the classes, can be varied on a range defined in percentage of the original value. The granularity of the execution is defined by the number of steps requested.

IMMAGINE DEL SERVICE TIMES SELEZIONATO PER UNA CLASSE, DAL 100% AL 200% IN 20 PASSI

The arrival rate of one (all) open class may be varied over an absolute (percentual) range that will be scanned in the specified number of executions.

IMMAGINE DELL'ARRIVAL RATE SELEZIONATO PER TUTTE LE CLASSI IN 10 PASSI DA 100% A 50%

The simulation seed cannot be changed over a range. The requested number of repeated run will be performed using a sequence of randomly generated seeds.
