Step 2 - Stations

_________________

 

The Stations tab allows you to define the stations that comprise the model.

 

1 - The number of stations of the model can be specified in the corresponding input area

 

 

either by adding one station at a time by clicking the button or you can add them all at once by selecting the desired number from the form .
For each station, name and type must be specified.
To delete a station, click the button .

 

 

2 - In order to personalize a model, stations may be given arbitrary names other than the default ones (i.e., Station1, Station2, etc.).

 

 

 

 

3 - Stations can be of any of the following types: queueing station, delay, fork, join, routing station.

 

If there at least an open class, two ad hoc stations are created by the system, namely a source and a sink station:

 

These stations are necessary for the simulation and cannot be removed.
