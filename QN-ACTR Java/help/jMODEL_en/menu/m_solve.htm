Solve menu commands

_________________________________

Simulate

Alt-S

This command is used to start the simulation of the queueing network. If you have built a correct model, this command will start the simulation immediately. If some errors are found, a popup window showing all problems will be shown.

See "Step 7: start the Simulation", "How to read the Simulation Results" or "How to correct error or warnings messages".

Export to JMVA

Alt-M

This command gives you the possibiliy to export the model you created with JSIMgraph, to JMVA. IIf some errors are found, a popup window showing all problems will be shown.

See "How to export in JMVA".

Show Results

Alt-R

After the simulation of the model and during the solution process, you can show the simulation results with this command.

See "How to read the Simulation Results".
