Define menu commands

___________________

Customer classes

It is used to define the customer classes of the queueing network model,

See "How to define Customers Classes".

Performance indices

This command is used to select the performance indices that will be evaluated and plotted by the Simulation Engine

See "How to define Performance Indices".

Simulation parameters

It is used to define the simulation parameters and the starting point of the system represented by the model.

See "How to define Simulation Parameters".

What-if analysis

This command is used to execute a what-if analysis, executing multiple simulation changing a parameter chosen as control variable.

See "How to define What-if analysis..."

Default parameters

This command is used to view and change default parameters and the initial default values of any object. See "How to change the Default Parameters"
