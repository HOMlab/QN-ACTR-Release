Modify Simulation Parameters

________________________________________

 

To modify the simulation or initial state, you have to select Simulation Parameters from Define menu and change the values.

 

From Simulation Parameters you can:

  1. change the Initial state, i.e., location of the customers at the beginning of the simulation

  2. change or fix the random engine seed number in order to have a different run of the simulation

  3. change the maximum number of samples

  4. change the time interval used in the graphical representation of the indices

  5. change or fix the maximun duration of the simulation

 

To see a complete help of simulation parameter see "Define Simulation Parameters"
