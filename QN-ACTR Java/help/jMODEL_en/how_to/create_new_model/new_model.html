Create a New Model

________________________

 

To draw a new network press or New from File menu.

JSIMgraph will show you a white layer on the principal panel.

 

The following steps should be followed:

 

  1. Draw the tolopogy if the network using the pre-defined objects (via click and drop): queue station, delay, source, sink, connection, routing station and fork/join.

  2. Define Customer Classes and select the Reference Station for each class

  3. Set the parameters for each object

  4. Choose and set the performance indices

  5. If you need, insert one or more finite capacity regions

  6. Choose or change the simulation parameters

  7. Enable What-if Analysis and set its parameters if required

  8. Start the simulation

  9. If diagnostic errors appear, click on them and the relative window will be shown
