Step 5: Define Reference Station

__________________________________________

 

For each class defined in Customer Class, a Reference Station should be set.

 

This station is used to estimate the system throughput for each class (i.e. number of jobs that flow through the system in a time unit). It is a required parameter in order to compute correctly teh simulation results. An error message will appear when the simulation is started without its prior definition

 

To set this parameter, see "How to define Customer classes".
