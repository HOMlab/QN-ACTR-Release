Example 2: Export as Gif

_________________________________

 

This example shows how to export a model as an image and it is particularly suitable to export the model in Gif format.

This is the model:

 

 

From Edit menu click the command "export as image..." and this form will appear in which you can select the exporting path and format, (select gif as in this example):

 

 

If you want a more accurate exporting, select the Options button near the format file, in this case the options for gif file are:

 

 

Select the option you want and press OK.

To save your image press OK again and JSIMgraph will save the image.
