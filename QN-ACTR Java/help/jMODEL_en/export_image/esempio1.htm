Example 1: Export as Pdf

_________________________________

 

When the model is complete, you may export it in .pdf format.

Below is an example of the model:

 

 

Press "Export as image..." from Edit menu, JSIMgraph will show you the panel where you should insert the path and choose the format of the file. In this example you can see the file name as "model1" in path "C:\", and the pdf format.

 

 

Now you can set the saving options. Click "Options" button and the following form will appear.

Remember that the options are different according to the type of format.

In pdf format there are these options:

 

 

In this case, some options are: page size, format, info like in a generic pdf file.

Enter your parameters and click OK, then click OK again and JSIMgraph will export the model in pdf file.
