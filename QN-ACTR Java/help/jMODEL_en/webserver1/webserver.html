Example 1: WebServer

_____________________________

 

In this example you can see the creation of simulation of WebServer..

To complete the model, follow the instructions step by step.

In this example is implements a webserver with one CPU and two Disks, each station property such as routing strategy of CPU, is explained in detail later.

Here is an exemple of What-If Analysis simulation:

  1. Implement the topology network

  2. Define customers classes and set the Reference Stations

  3. Define station parameters

  4. Define simulation indices

  5. Define simulation parameters

  6. Enable and set the What-if Analysis

  7. Start the simulation

  8. Read the results.
