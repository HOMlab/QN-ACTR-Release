Requisiti minimi.

Hardware:

  • Processore Intel PentiumII@300 o equivalente.
    Il simulatore è stato testato funzionante su AthlonXP2000+, PentiumIII@500MHz.
  • RAM 64MB (richiesti dalla JVM).
    Il simulatore è stato testato su sistemi con 128/256/512 MB di RAM.
  • 1MB di spazio su Hard Disk.
Software:

  • Sun Java Runtime Environment 1.4 o superiore.
    Il simulatore è stato testato su Windows2000/XP e Linux Debian/SuSE.
