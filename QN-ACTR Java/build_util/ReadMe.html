BEFORE YOU INSTALL JAVA MODELLING TOOLS...

It is recommended to UNINSTALL any previous version of Java Modelling Tools present on your PC before installing this new release.
New JMT versions will always be available, for free, at http://jmt.sourceforge.net .

Java Modelling Tools requires Java Runtime Environment (1.5 or higher). The Java Runtime Environment (JRE) provides the libraries, the Java Virtual Machine, and other components to run applets and applications written in the Java programming language.
For more info visit: http://java.sun.com/
