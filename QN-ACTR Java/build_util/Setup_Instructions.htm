JAVA MODELLING TOOLS: SETUP INSTRUCTIONS

  • To install and use Java Modelling Tools, you must accept the terms of license contained in the file "License.txt".


  • Java Modelling Tools includes third-party software that can be freely distributed according to the licenses contained in the folder "License".


  • Java Modelling Tools requires Java Runtime Environment.
    The Java Runtime Environment (JRE) provides the libraries, the Java Virtual Machine, and other components to run applets and applications written in the Java programming language.
    For more info and download, visit: http://java.sun.com/


  • If you want to use the shortcuts created by the JMT installer, please make sure that in your system .jar files have been correctly associated to the Java Virtual Machine. If .jar are associated to another application, you'll have to run Java Modelling Tools through command-line (using the command java - jar JMT.jar).


  • It is recommended to uninstall any previous version of Java Modelling Tools present on your PC before installing this new release.


Download

Download Java Modelling Tools at http://jmt.sourceforge.net/.
