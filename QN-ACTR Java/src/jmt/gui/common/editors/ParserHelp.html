This expression evaluator uses JFEP - Java Fast Expression Parser by Bertoli Marco. JFEP is designed to be lightweight and fast, suitable for multiple evaluations of the same function with minimum overhead. JFEP Home page: http://jfep.sourceforge.net

JFEP supports implicit multiplication, when a number in input string if followed by a character

Supported Operators (*)

Power ^
Unary Minus -n
Modulus %
Division /
Multiplication *
Addition, Subtraction +, -
(*) Ordered by operator precedences
